Reparacion-Service (puerto 8090, apr_reparacion_db)
Responsabilidad: Registrar reparaciones asociadas a incidencias. Al finalizar (FINALIZADA), actualiza la incidencia a RESUELTA. Entidad: Reparacion { id, incidenciaId, descripcionTrabajo, costoEstimado, estado: EstadoReparacion, fechaInicio, fechaFin, operadorId } Controller: ReparacionController — GET con filtros (incidenciaId, estado), POST, PUT, DELETE Service: ReparacionService — guardar(), actualizar(), buscarPorIncidenciaId(), buscarPorEstado(), obtenerIncidenciaDeIncidenciaService(), actualizarEstadoIncidenciaAResuelta() Repository: ReparacionRepository — custom: findByIncidenciaId(), findByEstado() DTOs: ReparacionDTO, ReparacionResponseDTO Dependencias externas: Incidencia-Service → GET /incidencias/{id} + PUT /incidencias/{id} (RESUELTA) Tabla BD: reparaciones
